## Features

- Professional dark design with violet/cyan palette
- Reveal animations on scroll (Intersection Observer)
- Fully responsive (mobile, tablet, desktop)
- Animated hamburger menu on mobile
- Animated counters in the "About me" section
- Scroll-to-top button with automatic appearance
- Active navigation that detects the visible section
- Optimized loading with lazy image loading
